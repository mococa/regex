# Regex úteis

### Dinheiro
^((R\$)?\s?\d.((,|.)\d.*)?)$

### Telefone
^(\(?\d{2}\)?(\9)?\d{4}(\-)?\d{4})$

### CEP
^(\d{5}\-?\d{3})$

### CPF
^(\d{3}\.\d{3}\.\d{3}\-\d{2}|\d{3}\d{3}\d{3}\d{2})$

### CNPJ
^(\d{2}\d{3}\d{3}\d{4}\d{2}|\d{2}\.\d{3}\.\d{3}\/\d{4}\-\d{2})$

### Horário 24h
([0-1]?[0-9]|2[0-4]):([0-5][0-9])(:[0-5][0-9])?

### Email
^[^\@\s\"\:\;]*\@\w*\.\w*(\.\w*)?$

### URL Link
